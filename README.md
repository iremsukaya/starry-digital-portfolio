# Irem's Starry World 🌟 | Personal Portfolio

Welcome to my personal universe! [cite_start]This project is a comprehensive web portfolio developed as part of my **MIS2006 Web Programming** course at **Halic University**. [cite: 1, 3]

## 🚀 Project Overview
[cite_start]This website is designed to showcase my journey as a 4th-year **Management Information Systems** student. [cite: 1, 4] It blends technical logic with a "starry" aesthetic to create a unique user experience.

### ✨ Key Features
- **Dynamic Star Engine:** A custom JavaScript function that populates the background with twinkling, animated stars on every page load.
- [cite_start]**Glassmorphism UI:** A modern design approach using frosted-glass effects, soft pastel gradients, and responsive layouts. [cite: 1, 15]
- [cite_start]**Interactive Greeting:** A personalized engagement feature where users can receive a starry welcome addressed to them. [cite: 1, 21]
- [cite_start]**Full Responsiveness:** Optimized for both desktop and mobile viewing. [cite: 1, 16]

## 🛠️ Tech Stack
- [cite_start]**Frontend:** HTML5, CSS3 (Custom animations & Flexbox) [cite: 1, 8, 13]
- [cite_start]**Logic:** Vanilla JavaScript (DOM Manipulation & Event Handling) [cite: 1, 18, 20]
- **Design:** Pastel Glassmorphism & Font Awesome Icons
