Irem's Starry World 🌟 | Personal Portfolio
Welcome to my personal universe! This project is a comprehensive web portfolio developed as part of my MIS2006 Web Programming course at Halic University.

🚀 Project Overview
This website is designed to showcase my journey as a 4th-year Management Information Systems student. It blends technical logic with a "starry" aesthetic to create a unique user experience.

✨ Key Features
Dynamic Star Engine: A custom JavaScript function that populates the background with twinkling, animated stars on every page load.

Glassmorphism UI: A modern design approach using frosted-glass effects, soft pastel gradients, and responsive layouts.

Interactive Greeting: A personalized engagement feature where users can receive a starry welcome addressed to them.

Full Responsiveness: Optimized for both desktop and mobile viewing.

🛠️ Tech Stack
Frontend: HTML5, CSS3 (Custom animations & Flexbox)

Logic: Vanilla JavaScript (DOM Manipulation & Event Handling)

Design: Pastel Glassmorphism & Font Awesome Icons
